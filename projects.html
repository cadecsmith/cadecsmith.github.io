Static Maps

Dominican Republic General Reference Map

As part of a cartographic design class at BYU, I created a general map of the Dominican Republic using ArcGIS Pro. This was one of my first attempts at a large general reference map that included physical and human features.

America's Rock Climbing Maps

I created several maps to map the spatial distribution of rock climbing in the United States, using several different variables and data from popular climbing website MountainProject. Uinta Wasatch Timber Harvest

I performed a suitability analysis for timber harvest in the Uinta Wasatch Cache National Forest. This project involved creating a map of suitable areas for timber harvest based on slope, aspect, and distance from roads.

Utah Resort Suitability Index

I created a suitability index for a hypothetical luxury resort in Utah based on several factors like distance to national parks and recreation areas, slope, and distance from roads. This project was a great way to practice spatial analysis and suitability modeling.

Aflatoxin in Corn Study

As a research assistant for a professor at BYU, I created a poster with several maps of statistical correlation between risk factors for aflatoxin in corn in northern Texas.

Web Maps

Personal Fishing Database

For my own personal use, I created a web map to log all fish I have caught along with information like the lure, time of day, weather, and size and species of the fish. This project was a great way to practice web mapping and database management.
